# PRD.md

# Automation Studio Portfolio MVP

## Goal

Build a premium portfolio website for an AI Automation Engineer that
looks like a modern SaaS product and converts visitors into qualified
message inquiries.

## Target Clients

-   HVAC
-   Restaurants
-   Local Service Businesses
-   US, UK, Canada

## Primary CTA

**Send a Message**

Supported: - Contact Form - Email - LinkedIn - WhatsApp (optional)

No "Book a Call" as the main CTA.

## Tech Stack

-   Next.js
-   TypeScript
-   Tailwind CSS
-   shadcn/ui
-   Framer Motion

## Design Inspiration

-   codewonders.dev
-   Linear
-   Vercel
-   Raycast

Style: - Minimal - Premium - Editorial - Large typography - Dark-first -
Lots of whitespace - Smooth animations
